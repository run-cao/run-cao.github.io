# VMware 上安装 Ubuntu

如果在 VMWare 中创建虚拟机并立刻开始安装 Ubuntu，VMWare 会很『贴心地帮你进行相关配置』。你只需要设置很少内容（如用户名、密码）即进入到了安装中的界面。

不过，对于初学者，我们可能还是想多见见、多练练安装系统过程中的相关配置。

这种情况下，你先创建一个空的虚拟机，不指定所安装的系统镜像，在创建成功之后，再指定光驱加载的 iso 镜像文件，随后再开始安装。
